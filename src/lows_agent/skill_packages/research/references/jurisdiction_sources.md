# 法域官方来源表

此表用于选择检索域名，不代表任何页面中的文件当然现行有效。仍须读取原文并核验时间与效力。

## 中国

- `flk.npc.gov.cn`、`npc.gov.cn`：国家法律法规、全国人大材料
- `gov.cn`：国务院法规及政府文件
- `moj.gov.cn`：司法部规章、规范性文件和官方说明
- `court.gov.cn`：最高人民法院司法解释、案例和司法材料
- `spp.gov.cn`：最高人民检察院司法解释性文件和案例材料
- `alk.12348.gov.cn`：司法行政法律服务案例，仅作案例或实务参考

## 美国

- `congress.gov`：联邦立法过程与法案
- `uscode.house.gov`：United States Code
- `govinfo.gov`：官方出版物、Public Laws、Federal Register 等
- `ecfr.gov`：Code of Federal Regulations
- `supremecourt.gov`、`uscourts.gov`：联邦法院材料
- 州法问题必须进一步确认州，并优先使用该州立法机关、政府或法院官方域名

## 欧盟

- `eur-lex.europa.eu`：欧盟立法、官方公报及相关文件
- `curia.europa.eu`：欧盟法院判决和程序材料
- 成员国国内法问题不等于欧盟法问题，必须另行识别成员国法域

## 英国

- `legislation.gov.uk`：英国成文法
- `supremecourt.uk`：英国最高法院判决
- `judiciary.uk`：英格兰和威尔士司法材料
- 必须区分英格兰和威尔士、苏格兰、北爱尔兰的法律体系

## 其他法域

优先查找该法域的官方立法机关、政府公报、司法机关和监管机构网站。域名后缀只能作为线索；`.gov`、`.gov.xx` 或政府风格页面本身不能证明文件真实、现行或具有特定效力。
